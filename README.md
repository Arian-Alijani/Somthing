## 📱 نحوه‌ی استفاده

۱. لینکِ اشتراکِ دسته‌ی دلخواه (ستون Base64) را کپی کنید.

۳. اشتراک را **Update** کنید تا کانفیگ‌ها بارگذاری شوند.
